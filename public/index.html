Badger API

/readUser

GET /readUser?id=2
Returns user with id 2 in JSON
Sample Output:
{
"id": 2,
"username" : "exampleUser1",
"email" : "someone@example.com",
"groupIds" : [1, 2, 3]
}


/createUser

POST /createUser
Returns user with id 4 in JSON
Sample Request Body:
{
"username": "myUser"
}
Sample Output:
{
"id": 16,
"username" : "myUser",
"email" : "someone@example.com",
"groupIds" : []
}


/readGroup

GET /readGroup?id=10
Returns group with id 10 in JSON
Sample Output:
{
  "id": 10,
  "groupName": "My Group",
  "groupDescription": "Description",
  "userIds": [
    26
  ]
}


/createGroup

POST /createGroup
Creates and returns group with id in JSON
Sample Request Body:
{
	"groupName" : "My Group",
	"groupDescription" : "Description"
}
Sample Output:
{
  "id": 13,
  "groupName": "My Group",
  "groupDescription": "Description",
  "userIds": []
}


/addUserToGroup

POST /addUserToGroup
Adds given user to given group
Sample Request Body:
{
	"userId" : 26,
	"groupId" : 7
}
Sample Output:
{
  "response": "User added to group.",
  "userId": 26,
  "groupId": 7
}
